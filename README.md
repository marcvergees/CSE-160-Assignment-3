Welcome to my **Blocky World**, a custom-built WebGL environment featuring a medieval castle theme. Explore the castle, interact with the specialized mechanisms, and enjoy the immersive medieval atmosphere.

### üåü Key Features

*   **Immersive Medieval World**: A castle environment built with varied block types and textures including stone, wood, trees, diamond blocks, chests, TNT, and terrain.
*   **First-Person Navigation**: Move freely through the world using standard WASD controls and mouse look.
*   **Performance Optimization**: Utilizes `renderfast()` for efficient rendering of the large block-based map and scene.
*   **Dynamic UI**: A custom-styled medieval user interface with controls for camera perspective, gate mechanisms, and debug visualization.
